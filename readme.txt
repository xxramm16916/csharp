Тестовое задание для ТНЕ.

TNE_v2 - серверная часть. Осуществляет первичную инициализацию при запуске, обрабатывает запросы от клиента.
Структура БД взята из тестового задания ТНЕ.

Bootstrap - клиентская часть. Предоставляет доступ к методам серверной части. Прошу обратить внимание, что вывод интерфейсной части внизу страницы.
